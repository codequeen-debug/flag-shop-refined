This project highlights an updated and refined version of MINI-PROJECT-ONE.
Our refined flag shop includes striking feautures like an add to cart, and subscription buttons as well as buttons that take you to additional connectivity with our team.
Namely, linkedIn, X, facebook and Youtube buttons.

## Subscription Logic
- Our Subscribe button includes an email validation feature which allows us to check user input to ensure accurate emails are provided to our news letter
- A pop up is provided once an acceptable email is provided.

## Cart Logic
- We have added a new and improved "Buy Now" button to our card layout.
- Our cart features a remove button as well as a total cost count.
